# Documentation

This directory contains all project documentation organized by category.

## Directory Structure

- **admin/** - Admin dashboard features and improvements
- **auth/** - Authentication and OAuth configuration
- **client/** - Client dashboard features and fixes
- **deployment/** - Deployment guides and production setup
- **features/** - Feature implementations and system documentation
- **fixes/** - Bug fixes and issue resolutions
- **integrations/** - Third-party integrations (Stripe, Vercel, GitHub, AVFY, etc.)
- **maintenance/** - Maintenance system and billing documentation
- **start-here/** - Getting started guides for developers
- **vercel/** - Vercel-specific configuration and integration
- **archive/** - Old files and legacy documentation

## Key Documents

- **PROJECT_STRUCTURE.md** - Overview of the project architecture
- **RECOMMENDATIONS.md** - Best practices and recommendations
- **SETUP_CHECKLIST.md** - Initial setup guide
- **ENV_VARS_CHECKLIST.md** - Environment variables configuration
